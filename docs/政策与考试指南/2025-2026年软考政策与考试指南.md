## 一、机考改革详情

### 改革时间线
| 时间 | 事件 |
|------|------|
| 2023年下半年 | 软考开始全面实行计算机化考试（机考） |
| 2024年 | 机考全面铺开，分批次考试 |
| 2025年 | 继续机考，多科目更新教材 |

### 机考形式
- **考试方式**: 计算机化考试（机考）、闭卷
- **科目连考**: 综合知识和案例分析/应用技术两科连考
- **高级考试（三科连考）**:
  - 综合知识 + 案例分析 = 240分钟总时长
  - 综合知识最长150分钟，最短120分钟
  - 论文单独安排（150分钟）
- **中级考试（两科连考）**:
  - 综合知识最短90分钟，最长120分钟
  - 两科总时长240分钟
  - 考试结束前60分钟可交卷离场
- **分批次考试**: 报考人数多的科目分多个批次

### 题型变化
- 综合知识：单选题（高项75题，中级75题）
- 案例分析：主观题+客观题混合
- 论文（高级）：2000-2500字

## 五、各省市积分落户政策

### 广州市

**现行政策**（2025年6月到期）:
- 软考高级 + 50周岁以下 + 在广州就业缴纳社保（1个月）→ 人才引进入户
- 软考中级 + 40周岁以下 + 在广州就业缴纳社保（1个月）→ 人才引进入户
- 外地证书需注意：取得时单位不在广州的，须在广州无社保记录

**征求意见稿**（未实施）:
- 高级职称年龄放宽至55周岁
- 中级职称年龄放宽至45周岁
- 来源: 《广州市户籍迁入管理规定》（修订稿）2024年11月发布

### 深圳市

根据（深人社规〔2016〕22号）:
- 软考高级 + 50周岁以下 + 中专以上学历 → 核准类人才引进入户
- 软考中级 + 45周岁以下 + 中专以上学历 → 核准类人才引进入户

### 北京市

根据《北京市引进人才管理办法（试行）》:
- 具有高级专业技术职称（含软考高级）可申请人才引进
- 积分落户中，高级职称可加分
- 软考证书可申请北京工作居住证

### 上海市

**方式一：居转户**
- 中级职称聘任条件：本科+5年相关工作经验 或 硕士+2年
- 高级职称聘任条件：本科+10年 或 硕士+7年 或 博士+2年
- 需满足岗位、社保缴纳等要求

**方式二：人才引进**
- 软考高级 + 受聘相应职务 + 在沪缴纳社保
- 一般距法定退休年龄5年以上

### 天津市

**方式一：积分入户**（政策有效期至2026年12月31日）
- 软考高级：+30分
- 软考中级：+20分
- 软考初级：+15分

**方式二：人才引进（海河英才）**
- 软考高级符合"资格型人才"中副高级职称条件

### 东莞市

（自2024年6月1日起）
- 软考初级 + 社保在缴 + 未满45周岁 + 中等教育以上学历
- 软考中级 + 社保在缴 + 未满50周岁 + 中等教育以上学历
- 软考高级 + 社保在缴 + 未满55周岁 + 中等教育以上学历

### 珠海市
- 软考中级/高级 + 男性50周岁以下（女性45周岁以下）→ 可申请

### 中山市
- 具备初级及以上职称即可申请人才入户

### 其他福利
- 部分省市提供持证补贴（金额各地不同）
- 可用于评定职称（以考代评）
- 企业资质申请加分

## 六、重要提醒

1. **报名渠道**: 中国计算机技术职业资格网 (https://www.ruankao.org.cn/)
2. **报名时间**: 一般考前2-3个月开放，具体以各地通知为准
3. **合格标准**: 各科目均需达到45分（满分75分）
4. **证书有效期**: 长期有效，无需年审
5. **单独划线地区**: 部分地区有单独的合格标准
