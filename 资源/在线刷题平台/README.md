# 在线刷题平台推荐

> 以下平台均经过验证，确认可用（截至2025年3月）

## 🆓 免费平台

### 1. 软考达人（微信小程序）

- **入口**: 微信搜索"软考达人"
- **覆盖科目**: 系统架构设计师、信息系统项目管理师、软件设计师等
- **题库规模**: 2009-2025年历年真题
- **特色**:
  - 完全免费
  - 支持错题本
  - 每日推题
  - 配合 xiaomabenten/system_architect 仓库使用
- **验证**: ✅ 2025-03-23

### 2. 51CTO 题库

- **地址**: [t.51cto.com](https://t.51cto.com/list/years/sub-240)
- **覆盖科目**: 全科目
- **特色**:
  - 分年份、分科目在线做题
  - 有答案和简单解析
  - 部分功能免费
- **验证**: ✅ 2025-03-23

### 3. 牛客网

- **地址**: [nowcoder.com](https://www.nowcoder.com)
- **覆盖科目**: 部分IT类科目
- **特色**: 社区讨论活跃

## 💰 部分付费平台

### 4. 希赛网

- **地址**: [educity.cn](https://www.educity.cn/rk/)
- **覆盖科目**: 全科目
- **特色**:
  - 真题 + 模拟 + 视频课
  - 有考前冲刺直播
  - 部分内容需VIP
- **价格**: 免费注册，VIP约200-500元/年

### 5. 环球网校

- **地址**: [hqwx.com](https://www.hqwx.com)
- **覆盖科目**: 全科目
- **特色**: 真题练习 + 考前模拟

## 💡 刷题建议

1. **优先级**: 先刷近3年真题，再扩展到近5年
2. **频率**: 每天至少 20 题，考前一个月加到 50 题/天
3. **错题复习**: 建立错题本，考前集中复习
4. **时间控制**: 综合知识每题不超过 2 分钟
5. **分析考点**: 每道题标注考点，方便统计薄弱项
