## 六、使用建议

### 下载优先级

1. **GitHub/Gitee 仓库** — 直接 clone，最方便
2. **官方免费资源** — 凤凰架构 PDF、DDIA 在线版
3. **微信读书** — 免费阅读，无需下载
4. **脚本之家/书葵网** — 需要注册，有下载限制
5. **CSDN 文库** — 需要积分，质量参差不齐

### 注意事项

- 官方教材建议购买正版（考试命题依据）
- GitHub 上的 PDF 可能随时被 DMCA 删除，建议及时保存
- 扫描版 PDF 一般较大（30-100MB），OCR 版较小但可能有文字错误
- 部分 Gitee 仓库是 GitHub 镜像，国内访问更快
