## 五、购买渠道

| 渠道 | 优势 | 注意事项 |
|------|------|----------|
| **京东自营** | 正版保障、物流快 | 推荐首选 |
| **当当网** | 图书品类全、常有满减活动 | 注意选自营 |
| **天猫旗书官方旗舰店** | 出版社直营 | 正版保障 |
| **拼多多百亿补贴** | 价格低 | ⚠️ 注意辨别正版 |
| **二手平台（闲鱼/转转）** | 价格最低 | 适合买真题解析类 |
| **微信读书/豆瓣阅读** | 电子版方便 | 部分教材有电子版 |
| **清华大学出版社官网** | 出版社直销 | 可确认最新版次 |

### 购买建议

1. **官方教材必买正版新版** — 这是考试命题依据
2. **真题解析强烈建议购买** — 比看教材更高效
3. **辅导书选1-2本即可** — 不要贪多
4. **技术深化书按需购买** — 有余力再看
5. **善用图书馆和电子资源** — 大学图书馆通常有软考教材
