## 📢 关于软考

**计算机技术与软件专业技术资格（水平）考试**（简称"软考"）是由国家人力资源和社会保障部、工业和信息化部领导下的国家级考试。

- **考试性质**：国家职业资格考试，全国统一大纲、统一试题、统一评分
- **考试频次**：每年上半年（5月）和下半年（11月）各一次
- **考试形式**：2024年起全面实行机考
- **证书价值**：
  - 🏅 以考代评——直接获得对应级别的职称资格
  - 🌍 国际互认——与日本、韩国等多国实现互认
  - 🏙️ 积分落户——多地纳入人才引进积分体系
  - 💼 招投标加分——企业资质评定加分项

### 考试级别与科目

| 级别 | 代表科目 | 考试科目 |
|------|----------|----------|
| **高级** | 系统架构设计师、信息系统项目管理师、系统分析师、网络规划设计师 | 综合知识 + 案例分析 + 论文 |
| **中级** | 软件设计师、网络工程师、信息安全工程师、数据库系统工程师、系统集成项目管理工程师 | 综合知识 + 应用技术 |
| **初级** | 程序员、网络管理员、信息处理技术员 | 综合知识 + 应用技术 |
