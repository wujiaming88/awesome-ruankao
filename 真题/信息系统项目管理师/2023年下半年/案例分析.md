## 第一批次（10月28日）

### 试题一：采购与合同管理（25分）

**主要问题：**

1. 在采购管理方面做的有哪些不足？
2. 设立子合同需要满足什么条件？
3. 如果有外包的部分，供应商需要满足什么条件？需要满足什么条件才能分包？

**参考答案要点**：

**问题1 - 采购管理不足**：
- 没有制定采购管理计划
- 没有对供应商进行充分的资格审查和评估
- 没有建立有效的供应商绩效考核机制
- 合同条款不够完善，缺少违约责任约定
- 没有做好采购监控工作

**问题2 - 设立子合同的条件**：
- 必须经发包方书面同意
- 子合同的内容不得超出主合同的范围
- 总承包方仍需对发包方负总责
- 子合同的条款应与主合同一致或更严格

**问题3 - 分包条件**：
- 供应商需要具备相应的资质和技术能力
- 分包不能超过合同总量的一定比例
- 核心技术和关键工作不得分包
- 分包需经过业主方的书面同意
- 分包商需具备独立完成分包任务的能力

### 试题二：挣值管理计算题（25分）

经典的挣值管理计算题，题目较简单，不需要画图。

**考点**：PV、EV、AC、CV、SV、CPI、SPI、EAC、ETC计算

### 试题三：干系人与沟通管理（25分）

**主要问题：**

1. 背景中的项目经理在干系人管理与沟通管理有什么不足？
2. 管理干系人参与、监督干系人参与的定义与作用？
3. 背景中的项目经理做了哪些工作是符合干系人管理的？
4. 促进干系人参与的步骤？

**参考答案要点**：

**问题2 - 定义与作用**：
- **管理干系人参与**：与干系人进行沟通和协作，以满足其需要与期望，解决实际出现的问题，并促进干系人合理参与。主要作用是让项目经理能够提高干系人的支持，并尽可能降低干系人的抵制。
- **监督干系人参与**：监督项目干系人关系，并通过修订参与策略和计划来引导干系人合理参与项目。主要作用是在项目进展和环境变化时，维持或提升干系人参与活动的效率和效果。

**问题4 - 促进干系人参与的步骤**：
1. 识别干系人
2. 了解干系人的需求和期望
3. 分析干系人的影响力和利害关系
4. 制定干系人参与策略
5. 执行干系人参与策略
6. 监控干系人参与效果并持续改进

### 试题二：计算题 — 三点估算 + 关键路径 + 挣值（25分）

#### 题目描述

项目有A、B、C、D、E、F共6个活动，各活动给出了乐观时间、最可能时间和悲观时间。

#### 问题1（6分）

根据给出的乐观、悲观、最可能时间，用β算法算出各活动工期。

**参考答案**：
- 活动A的工期 = (1 + 3 + 2×4) / 6 = **2天**
- 活动B的工期 = (1 + 5 + 3×4) / 6 = **3天**
- 活动C的工期 = (0.8 + 6 + 2.8×4) / 6 = **3天**
- 活动D的工期 = (5 + 9 + 7×4) / 6 = **7天**
- 活动E的工期 = (1 + 3 + 2×4) / 6 = **2天**
- 活动F的工期 = (2 + 6 + 4×4) / 6 = **4天**

> 三点估算公式：工期 = (乐观 + 悲观 + 4×最可能) / 6

#### 问题2（4分）

算关键路径和总工期。

**参考答案**：
- 关键路径：**A → D → E → F**
- 总工期：2 + 7 + 2 + 4 = **15天**

#### 问题3（8分）

1. 如果活动C延迟1天，关键路径和工期会有什么变化？
2. 如果活动C延迟2天，关键路径和工期会有什么变化？

**参考答案**：
1. 活动C延迟1天：总工期不变（15天），关键路径变为**ABCEF和ADEF两条**
2. 活动C延迟2天：总工期变为**16天**，关键路径变为**ABCEF**

#### 问题4（4分）

如果每天每个活动的花费是2万，当项目到第9天末，总成本花费32万，此时A、B、C、D活动均完成，E活动完成一半，求项目CV、SV，说明绩效情况。

**参考答案**：
- PV = (2 + 3 + 3 + 7) × 2 = **30万元**
- EV = (2 + 3 + 3 + 7 + 1) × 2 = **32万元**
- AC = **32万元**
- SV = EV - PV = 32 - 30 = **+2万元**（进度超前）
- CV = EV - AC = 32 - 32 = **0万元**（成本持平）

**结论**：项目进度超前，成本持平。

#### 问题5（3分）

如果此时项目的偏差为临时性偏差，求项目EAC。

**参考答案**：
- BAC = (2 + 3 + 3 + 7 + 2 + 4) × 2 = **42万元**
- ETC = BAC - EV = 42 - 32 = **10万元**（临时性偏差用非典型公式）
- EAC = AC + ETC = 32 + 10 = **42万元**
