## 重要说明

### 系统分析师考试特点
- **2021年及以前**：每年1次，上半年5月考试，纸笔考试
- **2023年起**：改为机考，题量增至75题
- **2024年起**：每年2次（上半年5月+下半年11月）

### 考试科目说明

| 科目 | 题型 | 时长 | 合格线 |
|------|------|------|--------|
| 综合知识 | 75道选择题 | 150分钟 | 45分 |
| 案例分析 | 5道主观题（1必答+4选2） | 90分钟 | 45分 |
| 论文 | 4道论文选1道 | 120分钟 | 45分 |

三科均须≥45分方可通过。
