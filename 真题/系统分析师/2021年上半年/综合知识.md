## 综合知识考点范围

| 知识领域 | 核心考点 |
|----------|----------|
| 计算机组成与体系结构 | CPU组成、存储体系、总线、RISC/CISC |
| 操作系统 | 进程管理、内存管理、文件系统、死锁 |
| 数据库 | 关系代数、SQL、规范化、事务管理 |
| 计算机网络 | OSI/TCP模型、协议、网络安全 |
| 软件工程 | 开发模型、需求分析、设计模式、测试 |
| 面向对象 | UML图、设计模式、面向对象分析设计 |
| 系统分析与设计 | 结构化分析、数据流图、信息系统规划 |
| 信息安全 | 加密算法、认证、网络安全 |
| 知识产权与法规 | 著作权、专利权、商标权 |
| 数学与经济 | 概率、图论、运筹学 |
| 专业英语 | 英文阅读理解（5道题） |
