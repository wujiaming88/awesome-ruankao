## 2024年下半年（11月）综合知识真题

> 声明：整理网友回忆版，答案仅供参考，非官方标准答案。

**1.** CPU中不包含下列哪个（）？  
A 程序计数器  B 地址寄存器  C 地址译码器  D 指令寄存器  
**参考答案：C**

**2.** 访问内存1000次，Cache未命中50次，命中率是多少？  
A 5%  B 95%  C 5.5%  D 9.5%  
**参考答案：B**

**3.** 以下磁盘寻道算法，哪种算法会产生饥饿现象？  
A 扫描算法  B 最短寻道  C 先来先服务  D 电梯算法  
**参考答案：B**

**4.** 下列行为哪个不会引起系统中断？  
A 除数为0  B 键盘输入  C 堆栈溢出  D 访存缺页  
**参考答案：A**

**5.** 分区管理最佳分配算法，地址空闲按（）的次序登记空闲区表中？  
A 地址递增  B 长度递增  C 地址递减  D 长度递减  
**参考答案：A**

**6.** 分页文件的存储模式，说法正确的是？  
A 全部进程共享页空间，但只有运行的才允许放在页里  
B 全部进程共享页空间，所有的都放在页里  
C 每个进程单独一个页空间，进程下所有线程共用  
D 每个进程单独一个页空间，进程下各线程又有各自的  
**参考答案：A**

**7.** 目前图像识别用（）技术最合适。  
A 随机森林  B 决策树  C CNN卷积神经网络  D 向量机  
**参考答案：C**

**8.** 关于进程与线程，说法错误的是？  
A 线程不能并发，进程可以并发  
B 线程创建和销毁的开销较大，进程创建和销毁的开销较小  
C 线程共享同一进程的资源和内存空间  
**参考答案：A**

**9.** 事务执行失败后会怎样？  
A 全部回滚  B 部分回滚  C 全部提交  D 部分提交  
**参考答案：A**

**10.** 两个事务竞争资源出现死锁，怎么解决？  
A 提高事务优先级  B 其中一个事务回滚  C 事务超时立即重试  D 所有事务按相同顺序申请资源  
**参考答案：B**

**11.** 甲机器三台，乙机器2台，甲次品率0.3%，乙次品率0.2%，某一天质检员抽到1个次品，则是甲机器生产的概率。  
A 40%  B 50%  C 60%  D 70%  
**参考答案：D**  
> 解析：贝叶斯定理。P(甲|次品)= 3×0.3% / (3×0.3%+2×0.2%) = 0.9/(0.9+0.4) ≈ 69.2%

**12.** 某企业为了应对流量增长需快速扩展应用，通过哪种服务好？  
A 私有云  B PaaS  C IaaS  D SaaS  
**参考答案：C**

**13.** ERP中（）不是它的核心管理模块。  
A 客户关系管理  B 财务管理  C 人力资源管理  D 物流管理  
**参考答案：D**

**14.** （1）耦合-多个模块都访问同一个公共数据环境。（2）内聚-处理元素相关且按特定次序执行。  
(1) A 公共耦合  B 内容耦合  C 数据耦合  D 标记耦合  
(2) A 过程内聚  B 顺序内聚  C 逻辑内聚  D 时间内聚  
**参考答案：(1)A  (2)B**

**15.** 寄存器、主存、辅存、Cache访问顺序从快到慢是：  
A 寄存器-cache-主存-辅存  B 寄存器-cache-辅存-主存  
**参考答案：A**

**16.** 基于TCP的协议，能发送电子邮件的协议是？  
A FTP  B DNS  C SMTP  D HTTP  
**参考答案：C**

**17.** TCP/IP结构模型（1）处于最底层，（2）解决了路由选择的问题。  
**参考答案：(1)D网络接口层  (2)C网络互联层**

**18.** 212.32.15.18和212.33.16.31两个IP地址要位于同一子网，子网掩码是多少？  
A 255.255.255.0  B 255.255.0.0  C 255.0.0.0  D 255.255.255.254  
**参考答案：C**

**19.** IPsec中的（）技术可以防止sniffer监测。  
A IP包过滤  B 双向认证和共享密钥  C 封装安全有效的负载  D 身份交换和认证机制  
**参考答案：C**

**20.** 将模拟信号转换为数字信号的过程不包括？  
A 采样  B 量化  C 信源编码  D 信道编码  
**参考答案：D**

**21.** 大数据实时处理用什么工具最适合？  
A HADOOP  B SPARK  C Cassandra  D MapReduce  
**参考答案：B**

**22.** 云计算不包括？  
A PaaS  B SaaS  C HaaS  D IaaS  
**参考答案：C**

**23.** 产生死锁的四个条件？  
A 互斥+请求和保持+不可抢占+环路  
**参考答案：A**

**25.** 甲公司推出游戏产品，以下哪种做法最容易侵犯著作权？  
A 使用相近名字  B 使用甲公司的资源UI美术资源  C 使用相似画面和关卡  D 使用相似奖励方式  
**参考答案：B**

**26.** Yi+1=aXi+(1-a)Yi，其中0<a<1，属于什么算法？  
A 线性回归  B 指数平滑  C 折中准则  D 移动平均  
**参考答案：B**

**30.** 在请求分页系统中，处理顺序为？  
A 中断→决定淘汰页→页面调出→页面调入  
**参考答案：A**

**33.** 白盒测试覆盖最弱的是？  
A 语句覆盖  B 条件覆盖  C 路径覆盖  D 判定覆盖  
**参考答案：A**

**35.** 系统设计顺序是：  
A 数据设计→架构设计→人机界面设计→过程设计  
B 架构设计→数据设计→过程设计→人机界面设计  
**参考答案：B**

**39.** 软件开发模型中，（1）严格区分阶段，（2）快速开发原型基础上修改，（3）软件复用面向对象。  
**参考答案：(1)瀑布模型  (2)演化模型  (3)喷泉模型**

**43.** UML图中，类图描述了（1），对象图描述了（2）。  
**参考答案：(1)A一组对象、接口、协作和它们之间的关系  (2)A某一时刻某个对象及它们之间的关系**
