## 综合知识考点范围

根据多源信息汇总，2023年上半年系统分析师综合知识涵盖以下知识领域：

| 知识领域 | 预估分值 | 核心考点 |
|----------|----------|----------|
| 计算机组成与体系结构 | ~8分 | CPU组成、存储体系、总线、RISC/CISC |
| 操作系统 | ~6分 | 进程管理、内存管理、文件系统、死锁 |
| 数据库 | ~6分 | 关系代数、SQL、规范化、事务管理 |
| 计算机网络 | ~6分 | OSI/TCP模型、协议、网络安全 |
| 软件工程 | ~12分 | 开发模型、需求分析、设计模式、测试 |
| 面向对象 | ~8分 | UML图、设计模式、面向对象分析设计 |
| 系统分析与设计 | ~10分 | 结构化分析、数据流图、信息系统规划 |
| 信息安全 | ~5分 | 加密算法、认证、网络安全 |
| 知识产权与法规 | ~4分 | 著作权、专利权、商标权 |
| 数学与经济 | ~3分 | 概率、图论、运筹学 |
| 专业英语 | ~5分 | 英文阅读理解（5道题） |
| 新技术 | ~2分 | 云计算、大数据、人工智能、区块链 |
